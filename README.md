# Monthly Revenue Trend Analysis

This project contains a professional-grade Seaborn visualization for analyzing monthly revenue trends across different customer segments.

**Email:** 23f2004208@ds.study.iitm.ac.in

## Project Overview
Hagenes Hilpert - Customer Analytics Dashboard
Monthly Revenue Trend Analysis for Retail Client

## Files
- `chart.py` - Python script that generates the Seaborn lineplot visualization
- `chart.png` - Generated 512x512 pixel chart showing monthly revenue trends

## Requirements
- Python 3.x
- seaborn
- matplotlib
- pandas

## Usage
Run the following command to generate the chart:
```bash
python chart.py
```
